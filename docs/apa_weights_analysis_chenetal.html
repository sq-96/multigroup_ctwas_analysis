Analysis

Settings

  • Fusion Model: enet model, all SNPs with non-zero weights
  • thin = 0.1,
  • niter_prefit = 3
  • niter = 30
  • L = 5
  • group_prior_var_structure = “independent”
  • maxSNP = 20000
  • min_nonSNP_PIP = 0.5
